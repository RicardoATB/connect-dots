**Descripton**: Program that connects dots from a list of coordinate points. When the user click the plot, it displays the dots.

**Input**: Text file with a list of coordinates separated by space

Example for "flor.xt":
`
3 3 
1 2
0 0
-1 2
-3 3
-2 1
...
`
